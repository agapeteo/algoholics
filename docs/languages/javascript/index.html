JavaScript Array

There two array implementation approaches in JavaScript:

  • Dense arrays are like auto resizing arrays (dynamic array)
  • Sparse arrays are more similar to hash maps

JavaScript engines choose array implementation in run-time in most cases

create array example:

var arr = [0, 1, 2];
var anotherArray = new Array("a", "b", "c");

create empty array of specified length:

const arrLength = 5;
const arr = Array(arrLength).fill({});
// or
const arr = Array(arrLength).fill(null);

Map - associative array

JavaScript objects historically used as a maps. It's implementation also depends on browser engine, but most cases it look like hash based algorithm, so get() and set() is O(1)

EcmaScript 6 (sometimes called ES2015) and later versions have Map data structure, which similar to JS objects, but has additional features. You can read about Map and objects difference here

basic Map usage

const m = new Map();
m.set("one", 1); // set valur
m.get("one") // 1 - read value
m.has("one") // true - check if key exist
m.keys()

// using object 
const obj = {};
obj['one'] = 1;
obj.one; // 1
