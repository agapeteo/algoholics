Algorithm complexity - Big O

  • Constant O(1) -> associative array lookup, statements
  • Logarithmic O Log(N)-> binary search
  • Linear O(N) -> find maximum
  • Linearithmic O N log(n) -> quick sort
  • Quadratic O(N^2) -> check all pairs
  • Cubic O(N^3) -> check all triples
  • Exponential O(2^N) -> check all subsets
  • Factorial O(N!) -> generate all permutations brute force
